# Handbook

How the BetterAngels tech team operates: the principles we use to make
decisions and the context that shapes them.

## Contents

- [`principles/`](./principles/) — guiding principles, the format they follow,
  and the definition of the "context" that determines which principles win
  when they conflict.

## Public vs. internal

This handbook is intentionally public. Publishing how we make decisions is a
forcing function for clarity and a signal to partners, funders, and
contributors.

Some artifacts — most notably the **active context** for a given product or
for the org — may be kept internal when they reference specific partners,
blockers, or unreleased strategy. The framework is public; the current state
is not always.
